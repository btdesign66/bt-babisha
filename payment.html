Payment details
Order ID will be generated automatically (non-sequential, < 21 chars, alphanumeric).
We send this as x-routing-id (required by HDFC for affinity).
Note: We do not use udf2 (blocked for tokenization at bank end).
How it works
  1. We create an order on SmartGateway using your amount + customer ID.
  2. You’re redirected to HDFC’s payment page.
  3. After payment, HDFC redirects back to our response page.
  4. Response page fetches Order Status API and shows: Order number, Amount, Success message in real-time.
Customer ID auto-generation
If you don’t have a logged-in user id, we generate and store one in localStorage so it stays consistent per customer/browser.
